## 🎯 核心功能

### 1. 智能解析用户请求
- 自动识别课程名称、章节、题型、难度等参数
- 支持自然语言输入
- 智能匹配教材文件

### 2. 知识点提取
- 从教材中提取结构化知识点
- 标注章节和页码
- 建立知识点关联网络

### 3. 试卷生成
- **A卷**: 基础题占比≥70%
- **B卷**: 提升题占比≥70%
- 确保知识点覆盖率≥90%
- 自动去重和难度平衡

### 4. 质量验证
- 答案准确性检查
- 知识点覆盖验证
- A/B卷题目去重
- 章节平衡检查

### 5. 标准化输出
- Markdown 格式试卷
- 完整的参考答案
- 详细的评分标准
- 知识点统计表
