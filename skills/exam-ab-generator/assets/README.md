# Exam AB Generator - 使用说明

## 概述

`exam-ab-generator` 是一个基于 OpenClaw 的智能试卷生成技能，可以自动生成A/B两份试卷，包含标准答案和评分标准。

## 功能特性

- ✅ 基于教材/课程材料自动生成试卷
- ✅ 支持A/B两卷，题目不同但难度相当
- ✅ 自动生成标准答案和评分标准
- ✅ 支持多种题型：单选、多选、填空、简答、综合
- ✅ 智能知识点分布，确保覆盖度≥90%
- ✅ 标注教材章节和页码，便于核查

## 使用方法

### 基础用法

```
用户：为《Python程序设计》第3-5章生成期末试卷AB两卷

助手：正在生成试卷...
      [进度] 分析教材知识点...
      [进度] 生成A卷...
      [进度] 生成B卷...
      [进度] 生成参考答案...

      ✅ 试卷生成完成
      输出文件：
      - output/exams/python-basic/期末考试-A.md
      - output/exams/python-basic/期末考试-B.md
      - output/exams/python-basic/期末考试-answers.md
```

### 高级用法

#### 指定题型分布

```
用户：生成数据结构期中试卷，选择题30分，填空题20分，简答题30分，综合题20分
```

#### 指定难度

```
用户：为计算机网络生成一份较难的AB卷
```

#### 自定义配置

```
用户：使用配置文件生成试卷：assets/configs/python-exam-config.json
```

## 配置说明

### default-exam-config.json

默认配置文件，包含：

- **课程信息**: 课程名称、考试类型、总分、时长
- **题型设置**: 各题型数量、分值、难度分布
- **版本要求**: A卷/B卷的特殊要求
- **验证规则**: 知识点覆盖率、章节平衡等

### 自定义配置

创建自己的配置文件：

```json
{
  "course_name": "数据结构",
  "exam_type": "期末考试",
  "total_score": 100,
  "duration_minutes": 120,
  "difficulty": "困难",
  "chapters": [1, 2, 3, 4, 5],
  "question_types": {
    "single_choice": {
      "count": 20,
      "score_per_question": 2
    }
    // ... 其他题型
  }
}
```

## 输出格式

### 试卷格式

生成的试卷采用 Markdown 格式，包含：

- 考试信息（时间、总分、日期）
- 题型分类（单选、多选、填空、简答、综合）
- 每题标注参考知识点和教材位置

### 答案格式

答案文件包含：

- 每题的标准答案
- 详细解析
- 评分标准
- 知识点标注

### 知识点统计

自动生成知识点覆盖统计表，便于教师核查。

## 质量保障

系统自动进行以下检查：

- ✅ 答案准确性验证
- ✅ 知识点覆盖率检查（≥90%）
- ✅ A/B卷题目去重
- ✅ 章节平衡检查（避免单一章节占比过高）
- ✅ 教材引用完整性
